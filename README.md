## 프로젝트 소개

마법사를 캐릭터를 직접 생성하여 배틀 게임을 구현하였습니다.

## 프로젝트의 로직은 다음과 같습니다.
1. 캐릭터를 생성한다. (마법사 객체 생성)<br>
2. 선공을 고른다<br>
3. 나이별로 마법의 공격력을 계산한다.<br>
4. 선공자가 공격을 한다.<br>
5. 랜덤 데미지의 피해를 입는다.<br>
6. 대결의 승자를 출력한다.<br>
